01
Зачем это нужно
🚫Планирование съедает внимание каждый день.
Переписка «когда вам удобно», забытые встречи, перенос одной встречи тянет за собой три. У кого был живой помощник, тот платил ему в основном за ведение календаря.
💡ИИ-секретарь живёт в вашем Telegram и ведёт Google Календарь.

Вы говорите голосом «встреча с Иваном завтра в 15:00 по Москве». Встреча появляется в календаре со ссылкой на видеозвонок. Дальше он напоминает сам: за сутки, за час и за пять минут, прямо вам в личку.
Что вы говоритеЧто делает ИИ-секретарь
«Встреча с Иваном завтра в 15:00 по Москве»Создаёт событие, добавляет ссылку на созвон и показывает карточку на подтверждение
«Перенеси встречу с Иваном на пятницу на 20:00 по времени Москвы»Двигает встречу, длительность сохраняет
«Отмени встречу с Петей в четверг»Удаляет после подтверждения
«Какие встречи у меня на неделе?»Присылает список по дням
«Увеличь встречу с Иваном до двух часов»Меняет длительность
«Добавь во встречу описание и позови petya@mail.ru»Дописывает описание и отправляет приглашение на почту
«Переключи мой часовой пояс на Тбилиси»Меняет зону календаря, дальше считает всё в ней
«Какая у меня сейчас зона?»Показывает зону и текущее время
⚙️Работает само, без ваших команд:
• Утром присылает план дня
• В понедельник присылает план недели
• Напоминает за сутки, за час и за пять минут до встречи
• Ловит накладку и предлагает выбор: поставить всё равно, перенести или отменить
• Находит свободное окно, если попросить
• Считает часовые пояса и показывает обе зоны, когда встреча в другом городе
• Переспрашивает время, если вы его не назвали, вместо того чтобы придумать своё
✅Программный код руками писать не нужно. Вы отдаёте ссылку и промпт в Claude Code, дальше он всё делает сам и ведёт вас по шагам.

03
Доступ к Google: три вещи
⚠Это единственная часть, которую нужно сделать руками в браузере. Claude Code не может зайти в вашу консоль Google вместо вас.
Включить Calendar API
Откройте console.cloud.google.com, выберите свой проект, найдите Google Calendar API и нажмите «Включить».
Создать OAuth Client ID
В разделе «Учётные данные» создайте OAuth Client ID типа «Веб-приложение». Сохраните Client ID и Client Secret, они понадобятся Claude Code.
Перевести приложение в режим Production
На экране согласия переключите статус с Testing на Production.
🚫Если оставить режим Testing, доступ к календарю отвалится ровно через семь дней. Это самая частая причина, по которой секретарь замолкает через неделю.
04
Настройка секретаря
💡Всё в этом блоке делает Claude Code по промпту из блока 02. Ваша задача: подтверждать шаги и проверять результат.
Подключить Google Календарь
Claude Code подключает Google Workspace MCP и оставляет из него только Календарь.
Войти в свой Google-аккаунт по ссылке
Секретарь пришлёт ссылку. Открываете, входите, разрешаете доступ. Больше ничего вводить не нужно.
Задать правила секретаря
Часовой пояс, рабочие часы, длительность по умолчанию, обязательное подтверждение перед любым изменением.
Включить проверку накладок
Если новая встреча пересекается с существующей, секретарь предложит три варианта вместо того, чтобы молча поставить.
Включить работу с часовыми поясами
Секретарь считает время по названному городу и показывает обе зоны, когда они различаются.
Включить ссылку Google Meet
Добавляется к новым событиям автоматически.
Настроить напоминания
За сутки, за час и за пять минут до встречи, в личку.
Настроить план дня и план недели
План дня в 08:00, план недели по понедельникам в 09:00.
✅Правило подтверждения важнее, чем кажется. Без него секретарь однажды молча передвинет вам встречу с клиентом.
05
Zoom вместо Google Meet
💡Google Meet подключается сам и работает сразу. Этот блок нужен, только если вы проводите встречи в Zoom. Можно пропустить и вернуться позже.
Создать приложение в Zoom
Откройте marketplace.zoom.us, войдите под своим аккаунтом, раздел Develop, пункт Build App. Тип приложения: Server-to-Server OAuth. Такой тип работает без окна входа, это нужно для агента.
Выдать права на создание встреч
В разделе Scopes добавьте права на встречи: чтение и запись. Без них Zoom не даст создавать конференции.
Скопировать три ключа
На вкладке App Credentials лежат Account ID, Client ID и Client Secret. Скопируйте все три.
Активировать приложение
Вкладка Activation, нажмите Activate. Пока приложение не активировано, ключи не работают.
Передать ключи Claude Code
Отправьте три ключа в чат с Claude Code, он пропишет их в настройки секретаря сам.
⚠Client Secret это пароль от вашего Zoom. Не выкладывайте его в открытый репозиторий и не пересылайте в общие чаты.
06
Проверьте, что всё работает
🎯Отправляйте фразы боту одну за другой и отмечайте, что получилось. Голосом и текстом результат должен быть одинаковый.
«Поставь встречу с Иваном завтра в 15:00 по Москве»
Секретарь показывает карточку и ждёт подтверждения. После подтверждения событие появляется в Google Календаре.
Проверить ссылку на созвон
Откройте созданное событие: в нём должна быть ссылка на видеовстречу.
«Поставь встречу с Петей в четверг»
Время не названо. Секретарь обязан переспросить, а не выдумать время сам.
Поставить вторую встречу на то же время
Секретарь предупреждает о накладке и предлагает три варианта: поставить всё равно, перенести, отменить.
«Перенеси встречу с Иваном на пятницу на 20:00 по времени Москвы»
Встреча переезжает, длительность сохраняется.
«Увеличь встречу с Иваном до двух часов»
Меняется только длительность, остальное остаётся.
«Добавь во встречу с Иваном описание и позови petya@mail.ru»
Появляется описание, участнику уходит приглашение на почту.
«Какие встречи у меня на неделе?»
Приходит список, сгруппированный по дням.
«Какая у меня сейчас часовая зона?»
Секретарь называет зону и текущее время в ней.
«Переключи мой часовой пояс на Тбилиси»
Зона меняется. Следующее расписание приходит уже в новой зоне.
Поставить встречу в другом городе
Например «встреча в Дубае в 10:00». В карточке должны быть обе зоны: ваша и города встречи.
«Отмени встречу с Петей в четверг»
Секретарь переспрашивает и только потом удаляет.
Проверить напоминание
Поставьте встречу через десять минут и дождитесь напоминания за пять минут.
Проверить план дня
Не ждите утра: попросите Claude Code запустить рассылку вручную и убедитесь, что сообщение пришло.
