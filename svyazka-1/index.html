01
Что вы получите
💡Секретарь живёт в вашем Telegram и ведёт Google Календарь. Вы говорите голосом «встреча с Иваном завтра в 15:00», встреча появляется в календаре со ссылкой на видеозвонок. Дальше он напоминает сам.

Планирование съедает внимание каждый день. Перенос одной встречи тянет за собой три. У кого был живой помощник, тот платил ему в основном за это.
Что говоритеЧто делает секретарь
«Встреча с Иваном завтра в 15:00»Создаёт событие и добавляет ссылку на видеовстречу
«Перенеси Ивана на пятницу»Двигает событие и переспрашивает перед изменением
«Отмени встречу в четверг»Удаляет событие после подтверждения
«Что у меня на неделе»Отвечает списком по дням

02
Репозиторий и главный промпт
📦Весь код и готовые файлы лежат в репозитории. Вам не нужно ничего писать руками: вы отдаёте ссылку Claude Code, а он делает всё по инструкции.
Открыть репозиторий связки
Ссылку скопируйте, она понадобится в промпте ниже.
Репозиторий связки 1
⚠Промпт рассчитан на готовую базу из блока 03. Если чего-то не хватает, Claude Code остановится и скажет, чего именно.
📋 Промпт: собрать ИИ-секретаря (в Claude Code)
Ты настраиваешь мне личного ИИ-секретаря на OpenClaw. Работай по инструкции из репозитория: [УТОЧНИТЬ] Сначала проверь готовность и покажи мне результат каждой проверки: 1. Запусти openclaw status и openclaw doctor --fix 2. Проверь, подключён ли Telegram-канал и работает ли распознавание голосовых 3. Проверь, есть ли доступ к моему Google Календарю Если чего-то не хватает, остановись и скажи, что именно мне нужно сделать. Не придумывай обходные пути. Когда всё готово, продолжай: 4. Подключи Google Workspace MCP и включи из него только Календарь 5. Создай скилл секретаря: мой часовой пояс, рабочие часы, длительность встречи по умолчанию 6. Пропиши правило: перед созданием, переносом и отменой события всегда проговаривай, что именно делаешь, и жди моего подтверждения 7. Включи добавление ссылки на видеовстречу к новым событиям 8. Поставь два расписания: план дня в 08:00 и план недели по понедельникам в 09:00 После каждого шага показывай, что получилось, и жди моего «дальше».
Отправить промпт в Claude Code и получить результат первой проверки
Claude Code должен показать состояние сервера и список того, чего не хватает.
03
База: что должно быть готово
💡Всё из этого списка вы уже настраивали раньше. Здесь только проверка, что оно живо.
Сервер с OpenClaw работает
Подключитесь к серверу и выполните команду ниже. Обе команды должны пройти без красных ошибок.
openclaw status
Бот отвечает в Telegram
Напишите боту «привет» и дождитесь ответа.
Голосовые распознаются
Отправьте боту голосовое сообщение. В ответ он должен понять текст.
Доступ к Google на месте
Нужны три вещи: включённый Calendar API, созданный OAuth Client ID и приложение, переведённое в режим Production.
⚠Если приложение осталось в режиме Testing, доступ к календарю отвалится через семь дней. Это самая частая причина, по которой секретарь замолкает через неделю.
04
Настройка по шагам
Подключить Google Календарь
Claude Code делает это сам по промпту из блока 02. Проверить подключение можно командой ниже: она покажет список инструментов календаря.
openclaw mcp doctor --probe
Разрешить доступ к своему календарю
Скажите боту «подключи календарь». Он пришлёт ссылку, вы открываете её и входите в свой Google-аккаунт. Больше ничего вводить не нужно.
Задать правила секретаря
Часовой пояс, рабочие часы, длительность встречи по умолчанию и обязательное подтверждение перед изменением события.
✅Правило подтверждения важнее, чем кажется. Без него секретарь однажды молча передвинет вам встречу.
Включить ссылку на видеовстречу
Google Meet добавляется к событию сразу. Zoom и другие сервисы подключаются отдельно, это показано в видео.
Поставить план дня и план недели
Проверить, что расписания встали, можно командой ниже.
openclaw cron list
05
Проверьте, что всё работает
🎯Пять действий. Если все пять проходят, связка собрана.
Голосовое создаёт встречу
Скажите боту голосом: «поставь встречу завтра в 15:00, назови её проверка связки». Откройте календарь, событие на месте.
В событии есть ссылка на видеовстречу
Откройте созданное событие и убедитесь, что ссылка проставилась.
Перенос работает и требует подтверждения
«Перенеси проверку связки на пятницу». Секретарь должен переспросить до того, как двигать.
Отвечает на вопрос о расписании
«Что у меня на этой неделе». В ответе должен быть список по дням.
План дня приходит сам
Не ждите утра: попросите Claude Code запустить расписание вручную и проверьте, что сообщение пришло в Telegram.
06
Если что-то сломалось
Что видитеЧто делать
Секретарь не видит календарьВыполните openclaw mcp doctor --probe. Если инструментов календаря нет, попросите Claude Code подключить его заново
Работал неделю и пересталПриложение осталось в режиме Testing. Переведите его в Production и авторизуйтесь заново
Голосовые не понимаетВернитесь к чек-листу урока 3 и проверьте распознавание речи
Встречи встают не в то времяНе задан часовой пояс. Допишите его в правила секретаря
План дня не приходитВыполните openclaw cron list и проверьте, куда настроена доставка
💬Любую ошибку можно просто отправить в Claude Code целиком, вместе с текстом из терминала. Он разберёт и починит.
